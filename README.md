# FIFA World Cup 2026 — Match Schedule (UAE Time)

Interactive wall chart for the 2026 World Cup with all 104 matches in UAE time (GST, UTC+4).

- **[Animated version](index.html)** — video background
- **[Clean version](clean.html)** — static background

Features: full group stage + knockout bracket, tap any knockout slot to pick a country,
editable dates, country flags, auto-fullscreen on 16:9 displays (TVs).
